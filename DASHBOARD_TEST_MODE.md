# Urban Yards Dashboard Test Mode

Use this URL to view the dashboard with sample records:

https://urbanyards.us/dashboard?demo=1

Test mode does not use Supabase, Square, or real client data. The sample records only live in the current browser session and reset when the page is reloaded.

What to test:

- Today: metrics, quote pipeline, upcoming visits, reminders, and quick actions.
- Work: calendar views, add visit, edit visit, delete visit.
- Route Planner: change date, add stop, edit stop, reorder stops, mark complete, open route in Google Maps.
- Clients: open client detail, edit client, delete client, add client.
- Money: create estimate/invoice, open document, edit document, sync sample Square status, print/PDF.
- Operations: use templates, add operation record, filter records, delete record.
- Settings: add task, export CSV files, complete/delete reminders.

Because it is a browser-session demo, refresh the page to reset back to the original sample data.
